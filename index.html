about riga dev day

The conference was organised by Latvian Java User Group and Oracle at November 28th in Riga, Latvia. This was a major Baltics IT event welcoming more than 280 Java programmers from the region.

15 talks was presented by 15 speakers from Russia, Latvia, England, France, Greece, Czech Republic, Germany, Switzerland, Finland and Italy.

Photos and presentations from Java Day Riga 2013 are available now.

For any questions please contact Latvian JUG.
