About Riga Dev Day

Riga Dev Day is a software development conference organised by local Java User Group, Oracle User Group and Google Development Group. This social event is produced by software developers for software developers. All topics will be hands-on and beneficial for your daily job. And we have zero tolerance for sales pitches. Conference takes place in Multikino cinema on 22nd of January, 2015.

It is the first time this conference brand takes place in Riga, Latvia, accumulating three years of previous experience organizing similar events. This is going to be one of the largest IT events in Baltic States welcoming 400+ participants and 25+ internationally acclaimed speakers.

Talks will be around the following topics:

  • Software development on Java Virtual Machine
  • Oracle technologies (Oracle, MySQL, BI, etc.)
  • Web and Mobile development
  • DevOps and Continuous Delivery
  • Wearable technologies and embedded programming
